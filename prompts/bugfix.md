# Bug Fixing Template

## Bug: [DESCRIPTION]

## Symptoms
- What's happening
- - Expected behavior
  - - Actual behavior
   
    - ## Process
    - 1. Reproduce the bug
      2. 2. Identify root cause
         3. 3. Implement fix
            4. 4. Write regression test
               5. 5. Verify fix works
                  6. 6. Check no new issues
                    
                     7. ## Success Criteria
                     8. - Bug no longer reproducible
                        - - Regression test added
                          - - All existing tests pass
                           
                            - ## Stuck Protocol
                            - After 15 iterations if not fixed:
                            - - Document blocking issues
                              - - List attempted approaches
                                - - Output <promise>BLOCKED</promise>

                                ## Completion
                                Output <promise>FIXED</promise> when resolved.
